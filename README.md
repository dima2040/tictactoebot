# tictactoebot

Бот для игры в крестики-нолики.

## Установка и настройка

Создайте и активируйте окружение
```sh
python3 -m venv .venv
source .venv/bin/activate
```

Укажите нужную переменную окружения
```sh
export <bot_name>_API_TOKEN=***
```

Запустите бота
```sh
python <bot_name>.py
```
